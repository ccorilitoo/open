Options

You can customize the bahavior of Clarity by passing in options to the constructor

    
        var viewer = new DocumentViewer({
            $anchor: $('#full-width-container')
        });

Document Viewer properties that can be set in the documentViewerOptions property

Option Description Default Valid values
width The width of the document viewer 500 Integer
autoPlay Determines if audio/video files start playing immediately after they are loaded true Boolean (true/false)
autoPlay Determines if audio/video files start playing immediately after they are loaded true Boolean (true/false)
autoLoadDependencies Determines if the plugin should auto load the 3rd party plugins (pdf.js, jPlayer, flowplayer, prettify). If this is set to false, the dependencies have to be manually added to the page. true Boolean (true/false)
enableTextAndCode Determines if the plugin should load text and code file types (deprecated) false Boolean (true/false)
jPlayer This allows you to override any of the default jPlayer options. {} jPlayer options
timeoutValue Determines how much time to spend trying to load a document (milliseconds) 4000 Integer
path This is required for the dependency loader and tells the plugin where it is located relative to the file it is used in. 4000 Integer
