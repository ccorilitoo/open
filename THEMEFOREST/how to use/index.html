B) CSS Files - top

These are css files used in this template:
  1. bootstrap.min.css - css by Twitter boostrap
  2. animated-alice.css - used for element animation all of page
  3. alice-style.css - main css files (you can edit color, font, etc from this css)
  4. queries-alice.css - used for responsive style

You can change almost all styles from alice-style.css file.
